Librería de JavaScript: W3.JS

¿Qué es?

Librería de JavaScript diseñanada para simplificar el desarrollo WEB.

  • Fácil de aprender y aplicar
  • Orientada al desarrollo rápido de aplicaciones
  • Es multidispositivos
  • Gratutito y no requiere licencias

Se pueden realizar multitud de tareas en pocos minutos:

  • Realizar galería de imagenes con carrusel
  • Permite importar código HTML de otras webs
  • Ordenar el contenido de cualquier elemento HTML
  • Filtrar contenido de cualquier elemento HTML
  • Mostrar objetos de JavaScript en HTML
  • Leer datos de servidores Web
  • Ocultar y añadir clases a cualquier elemento
  • Añadir CSS a cualquier elemento HTML

¿Cómo añadir esta librería?

  1. Añadiendo un link al recurso en internet:
  2. Descargando el fichero de la red y utilizandolo con total libertad

Eliminar clases
